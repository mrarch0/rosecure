# Security Policy

## Supported Versions

The **main branch** is the live website and has no versions. Security fixes are applied directly.

The **staging branch** is private and only accessible to developers.

| Environment                  | Supported |
| ---------------------------- | --------- |
| Main branch                  | Yes       |
| Staging branch               | Yes       |
| Other copies or old versions | No        |

## Reporting a Vulnerability

Report any vulnerabilities to **[hello@rosecure.org](mailto:hello@rosecure.org)**.

Include a description and steps to reproduce the issue if possible.

Please do not share or exploit vulnerabilities. Valid reports may receive a bug bounty reward starting at $10, depending on severity.
